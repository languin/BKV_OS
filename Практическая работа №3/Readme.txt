Семейство процессов.
Процесс-родитель parent при запуске порождает N=argv[1] детей и
запускает в них код процесса child N-1 , с потомками,
но на одного меньше, чем предок. Порождение новых потомков завершится,
когда параметр потомка станет нулевым. Все процессы, кроме parent
спят 
   for(;;) sleep(1);
родитель может посылать управляющие сигналы (команды):
(a) SIGALRM - приказ вывести свой pid и список pid своих детей,
          затем детям рассылает SIGALRM по этому списку;
(k) SIGUSR1 - приказ разослать детям SIGUSR1, затем выполнить _exit(0);
(n) SIGUSR2 - приказ создать потомка с кодом child и argv[1] на единицу меньше своего N.
Дополнительные команды приведены в комментариях кода родителя.
