Практическая работа № 0.

"Файловая система и системные вызовы для работы с файлами и каталогами"

Основной материал по системным вызовам можно найти в книге Терренса Чана,

http://bkv.vvsu.ru/ftp/OSes/pub/BOOKS/Cpp4Unix.djvu (в локальной сети ВВГУ),

можно также пользоваться справкой типа "man 2 stat", встроенной в
систему Linux. Справка может быть на английском или русском языках в
зависимости от установки дистрибутива.


Задание:

Используя только системные вызовы (!) на языке С (или С++) 
написать программу, находящую среди регулярных файлов домашнего каталога
и всех его вложенных каталогов файлы, имеющие нулевой размер (то есть не
содержащие информацию), в том числе и "скрытые файлы". Скрытым файлом считается
файл, имя которого начинается с символа '.' (точка). Обычно такие файлы не
отображаютя при выводе спиcка файлов.

Использовать системные вызовы: open, chdir, stat, opendir, readdir.

Можно использовать функции для работы со строками (из <string.h> ).

Внимание!
Поскольку лучшим решением будет рекурсивный вызов функции при
обнаружении каталога среди файлов, нужно будет избегать "особых"
каталогов с именами "." и ".."! Эти каталоги нужно пропускать!
