Практическая работа №1
Написать программу, выводящую информацию о разделах на жестком диске.
Вывод сделать в понятном для неспециалиста виде (номер раздела,
название файловой системы, наличие флага загрузочности, начало
раздела на диске и его размер (в единицах LBA)).
В данном каталоге приведены копии загрузочных записей некоторых
жестких дисков с таблицей разделов msdos.
Использовать системные вызовы open, read, lseek. Для вывода
использовать библиотечные функции printf и fflush(stdout).
Проверять наличие маркера загрузочного сектора.
Ввод имени файла с данными через командную строку (int argc, char *argv[])
